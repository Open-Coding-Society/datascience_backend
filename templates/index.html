What should Data Science Students learn?

  1. Python is the foundation of our machine learning models and backend data workflows.
  2. Jupyter Notebooks are used for interactive, step-by-step ML experiments and visualizations.
  3. GitHub is essential for sharing, versioning, and collaborating on data science projects.
  4. Pandas and NumPy enable data analysis and manipulation at scale.
  5. scikit-learn and TensorFlow power predictive modeling and advanced ML techniques.

What is Flask? How do we deploy models?

  1. Flask lets us turn trained ML models into deployable APIs that anyone can use.
  2. Our projects use Flask Blueprints to keep model logic, API routes, and UI clean and modular.
  3. Students build full-stack apps that allow users to input values and get model predictions.
  4. We use Jinja2 to connect Python logic with live HTML pages for interactive feedback.

How do we manage Data & Models?

  1. Students use CSV files, SQLite, and JSON to store datasets and prediction results.
  2. Projects often integrate SQLite with Flask for persistent storage of input/output data.
  3. Model files are saved with joblib or pickle and loaded into Flask apps for inference.
  4. We also explore NoSQL tools like MongoDB to handle flexible data formats.
